Neural Network Model

A Long Short Term Memeory (LSTM) Neural Network was used to predict the weather. This is a type of recurrent NN, which are good for making predictions based on time series data. We chose LSTM over other types of neural networks because of it's feedback connection abilities. This will provide our model with greater accuracy and efficiency. After being trained with multiple years worth of data, a model within five degrees was created.

Go to page



All of our code is open source. Python and Flask were used to code the LSTM and the website. The actual weather data is collected from NOAA and predicted data are stored in SQLite Database. Our code is published on Github.
